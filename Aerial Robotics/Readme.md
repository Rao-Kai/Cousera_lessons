## 课程内容
本课程是Cousera专项课程的课程一

1. 介绍无人机的kinematic(运动学)和dynamic(动力学)

2. 实现对无人机的位置和姿态的PID控制

## 文档说明
assignment1 实现对无人机的1D控制(高度控制)

assignment2 实现对无人机的2D控制(跟踪二维平面轨迹)

assignment3 实现对无人机的3D控制，跟踪对象可为给定路径点后自主生成的轨迹
